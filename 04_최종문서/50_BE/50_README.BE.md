## 2. 마일스톤

| Gate | 목표 | Exit Criteria |
|---:|---|---|
| `BE-M0 계약 동결` | robot event·API·오류 계약 합의 | schema version과 idempotency 규칙 확정 |
| `BE-M1 더미 E2E` | 더미 robot→저장→조회 | event·media가 같은 ID로 조회됨 |
| `BE-M2 실제 이벤트` | HW/AI 실제 payload 수집 | schema contract test 통과 |
| `BE-M3 오프라인 복구` | retry·중복·순서 검증 | 누락 0, 중복 record 0 |
| `BE-M4 운영 조회` | patrol·event·device 상태 제공 | 권한 있는 사용자가 근거 조회·수정 가능 |
| `BE-M5 안정화` | backup·관측·장애 복구 | RC 장애·부하 시험 통과 |

## 3. 책임 경계

- 로봇의 주행 폐루프와 안전 정지를 소유하지 않는다.
- 결함 class·model 정확도를 소유하지 않는다.
- 수신한 원본 증거, 파생 분석과 담당자 수정을 구분해 보존한다.
