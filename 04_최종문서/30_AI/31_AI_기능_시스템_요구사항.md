## 2. 클래스 초기 정의

| 코드               | 의미                | 확정에 필요한 것   |
| ---------------- | ----------------- | ----------- |
| `SURFACE_WEAR`   | 표면 박리·돌기 손실·마모 후보 | 포함·제외 사례    |
| `CRACK`          | 선형 균열과 균열 집합      | 최소 크기·병합 규칙 |
| `BREAKAGE`       | 깨짐·결손·구조 손상       | crack과 경계   |
| `STEP_CANDIDATE` | 영상 기반 단차 후보       | 거리센서 확인 계약  |

## 3. 시스템 요구

- 결과에는 원본 frame ID·timestamp·model version·artifact hash를 포함한다.
- 낮은 confidence와 처리 오류를 정상 class처럼 반환하지 않는다.
- AI 결과는 담당자가 수정할 수 있고 원래 결과를 보존한다.
- 주행 안전 제어는 결함 AI 성공 여부에 의존하지 않는다.
- 입력 품질·latency·자원 초과를 health로 노출한다.
