# 01. 자율주행이란?

> ⏱️ 예상 읽기 시간: 5분
> 🎯 목표: **자율주행·자동화·원격조종의 차이**를 설명할 수 있다.

## 한 문장으로 설명하면

> **자율주행은 로봇이 센서로 주변을 살피고, 현재 상황을 판단한 뒤, 안전한 이동 방법을 스스로 결정하고 실행하는 기술이다.**

사람이 운전할 때 눈으로 길을 보고, 내 위치를 짐작하고, 어느 방향으로 갈지 판단한 뒤 핸들과 페달을 움직인다. 자율주행 로봇도 비슷한 과정을 컴퓨터와 센서로 수행한다.

## 🚗 비슷해 보이는 세 가지 방식

| 방식 | 누가 판단하는가? | 예시 | 예상 밖 상황 대응 |
|---|---|---|---|
| 🎮 원격조종 | 멀리 있는 사람 | 조이스틱으로 로봇 운전 | 사람이 직접 대응 |
| ⏰ 자동화 | 미리 작성한 순서 | 10초 직진 후 정지 | 정해진 조건만 대응 |
| 🤖 자율주행 | 로봇의 소프트웨어 | 길과 장애물을 보고 주행 | 센서 정보로 판단 |

핵심 차이는 **모터가 자동으로 움직이느냐**가 아니라, **현재 상황을 누가 보고 판단하느냐**다.

## 자율주행의 기본 순환

```mermaid
flowchart LR
    A["👀 인지<br/>주변에 무엇이 있지?"] --> B["📍 위치 추정<br/>나는 어디에 있지?"]
    B --> C["🧠 판단·계획<br/>어디로 가야 하지?"]
    C --> D["🛞 제어<br/>얼마나 움직이지?"]
    D --> E["🛡️ 안전 감시<br/>계속 가도 안전한가?"]
    E --> A
```

이 과정은 한 번만 실행되지 않는다. 로봇이 움직이는 동안 계속 반복되며, 새로운 장애물이나 경로 변화에 맞춰 명령을 수정한다.

## 우리 프로젝트에서는

걸음걸음 로봇의 목적은 단순히 목적지까지 빨리 가는 것이 아니다.

```mermaid
flowchart LR
    A["등록된 순찰 루트 선택"] --> B["점자블록 옆을 일정 거리로 주행"]
    B --> C["불량·장애물 관찰"]
    C --> D["위치와 사진 기록"]
    D --> E["루트 끝까지 안전하게 순찰"]
```

### 로봇이 해야 하는 판단

- 점자블록이 보이면 옆의 평행 경로를 유지한다.
- 점자블록이 가려지거나 끊겨도 등록된 루트를 참고한다.
- 장애물이 나타나면 감속·정지·우회 중 안전한 행동을 고른다.
- 우회한 뒤 원래 순찰 경로에 다시 합류한다.
- 센서나 컴퓨터 상태가 이상하면 주행보다 정지를 우선한다.

## 자율주행이라고 해서 무엇이든 가능한 것은 아니다

| 오해 | 실제 의미 |
|---|---|
| “AI가 있으면 어디서든 달릴 수 있다.” | 학습·시험한 환경과 안전 범위 안에서만 사용해야 한다. |
| “GPS만 있으면 길을 정확히 따라간다.” | 보도에서의 GPS 오차는 점자블록 폭보다 클 수 있다. |
| “카메라가 길을 한 번 찾으면 끝이다.” | 그림자, 낙엽, 사람, 흔들림 때문에 계속 다시 판단해야 한다. |
| “모델이 똑똑하면 안전장치는 필요 없다.” | AI와 별도로 비상정지·충돌 방지·watchdog이 필요하다. |

## 🛡️ 가장 중요한 원칙

> **모르겠으면 감속하고, 위험하면 정지한다.**

자율주행의 목표는 “무조건 계속 달리기”가 아니다. 로봇이 확신하지 못하거나 센서가 고장 난 상황에서 안전하게 멈추는 것도 올바른 자율주행 행동이다.

## 한 페이지 요약

- 자율주행은 **인지 → 위치 추정 → 판단·계획 → 제어 → 안전 감시**의 반복이다.
- 원격조종은 사람이 판단하고, 자동화는 정해진 순서를 따르며, 자율주행은 센서로 현재 상황을 판단한다.
- 걸음걸음 로봇은 점자블록 옆을 순찰하면서 불량과 장애물을 기록한다.
- AI가 있어도 독립적인 안전장치와 사용 범위가 반드시 필요하다.
