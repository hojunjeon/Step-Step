# 상위 Jetson
- 인지 - 판단 수행
- 이벤트 발생 시 서버에 이벤트 데이터(ED) 전송
  - 불량 점자블록 인식 : [이미지/Class(변색,크랙,벗겨짐,단차)/Bbox/Conf/좌표]
  - 장애물 인식 : [이미지/좌표]
    ** 어떤 장애물인지까지 분류하지는 않음
- 하위 보드에 목표 주행 명령 전송

## 센서 

### Camera 1: 자율주행용 전방 카메라
- 차선(점자블록) 추종 주행 판단 > 인식된 라인 하나에서 일정 거리 떨어진 지점을 추종 > 로컬 자율주행 차선
- 장애물 인식 트리거 발동 시 동작 > 화면 캡쳐 > 서버로 전송할 ED에 포함

### Camera 2: YOLO 인식용. 차량 측면에 지면과 평행하게 설치해 점자블럭을 수직으로 촬영할 수 있게 브라켓 등을 설치해 세팅
- YOLO로 [변색/크랙/벗겨짐/단차] 인식
- 불량 점자블록 인식 이벤트 트리거
  - 화면 캡쳐(bbox, class, conf) > 서버로 전송할 ED에 포함

### LiDAR
- 자율주행 정밀 판단
- 장애물 인식 이벤트 트리거 

### GPS
- 전체 경로 추종 판단 > 글로벌 자율주행 경로
- 이벤트 발생 시 글로벌 좌표 > 서버로 전송할 ED에 포함

# 하위 RaspberryPi
- 정밀 제어 수행
- 상위 보드에 센서피드백/구동계 상태 전송
- 상위 판단에 따라 상세 제어 명령 생성하고 수행
- 상위에 센서 피드백

## 센서

### 후륜 구동 모터 / 조향 모터
- 제어 명령에 따라 정밀 구동/조향

### IMU / 휠엔코더 
- 센서 피드백과 구동계 상태 데이터
