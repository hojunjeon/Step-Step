## 2. 이번 설계의 핵심 결정

### 2.1 컴퓨팅 구조

평가 일정과 지급 키트 활용도를 고려하여 **Jetson Orin Nano + Raspberry Pi 5의 2보드 구조**를 기본안으로 정한다.

- **Jetson Orin Nano**
  - Logitech Brio 100 카메라, YDLIDAR X4 Pro, USB GNSS 연결
  - YOLOv8n-seg, 점자블록 추종 영상 처리, 경로 계획, ROS 2 상위 제어
  - 서버 업로드 및 오프라인 큐 관리
- **Raspberry Pi 5**
  - 지급 Motor Driver HAT, PCA9685, GA25-370 모터, MG996R 조향 서보 제어
  - 휠 엔코더·IMU·하향 ToF 등 저수준 센서 수집
  - Jetson 명령 타임아웃 감시와 즉시 정지

두 보드는 지급 UTP 케이블로 직접 연결하고 고정 IP 기반 ROS 2 통신을 사용한다. Raspberry Pi는 하드 실시간 제어기는 아니므로, 로봇 속도를 낮추고 물리 비상정지·명령 타임아웃·독립 전원 차단을 함께 적용한다.

### 2.2 주행 방식

**사전 기록 경로를 전역 기준으로 사용하고, 점자블록을 지역 주행 기준으로 사용하는 하이브리드 방식**을 채택한다.

1. 정상 구간: 카메라가 추출한 점자블록 중심선을 따라 주행
2. 갈림길: 다음 사전 경로 방향과 가장 잘 일치하는 점자블록 가지 선택
3. 점자블록 단절: 오도메트리·IMU·LiDAR와 사전 경로를 사용하여 연결 구간 통과
4. 장애물: LiDAR 감지 → 정지 → 사진/좌표 저장·전송 → 저속 회피 → 경로 복귀

### 2.3 불량 분류 체계

평가용 지도 아이콘은 총 5종으로 고정한다.

1. **표면 마모·벗겨짐**
2. **크랙**
3. **파손**
4. **단차 후보**
5. **장애물**

`변색`은 조명·젖음·그림자 영향이 커서 평가 MVP에서는 독립 주행 클래스보다 **서버 VLM의 색상 대비·변색 속성**으로 처리한다. 데이터가 충분하고 재현성이 확인되면 후속 단계에서 독립 클래스로 승격한다.

### 2.4 서비스 데이터 단위

본 서비스는 민원 즉시처리 시스템이 아니다. 개별 아이콘을 쌓는 데서 끝내지 않고 다음 단계로 집계한다.

```text
원본 탐지 이벤트
→ 동일 순찰 내 결함 이벤트
→ 3~5 m 관리 구간
→ 보수 후보 공사 구간
→ 지자체 보수 계획
```

## 3. MVP 완성의 정의

평가일에 다음 흐름이 **연속 3회** 재현되면 MVP를 완성한 것으로 본다.

1. 사용자가 기기에서 루트를 선택한다.
2. 로봇이 60~100 m 상당의 축소·통제된 코스를 자율주행한다.
3. 정상 구간에서는 점자블록을 따라가고, 단절 또는 갈림길에서는 사전 경로로 진행한다.
4. 점자블록 불량 모형을 탐지해 대표 이미지를 한 번만 촬영한다.
5. 장애물을 발견해 정지·촬영·회피하고 원래 경로에 복귀한다.
6. 네트워크가 끊겨도 이벤트를 로컬 저장하고, 복구 후 중복 없이 업로드한다.
7. 서버 지도에서 5종 아이콘과 상세 이미지·위험도·추천 조치가 조회된다.
8. 인접 이벤트가 관리 구간으로 묶이고, 구간별 보수 우선순위가 표시된다.

## 4. 중요 제약과 확인 필요사항

- 제공된 GitHub 저장소는 문서 작성 시점의 비로그인 공개 환경에서 내부 파일을 확인하지 못했다. 따라서 기존 코드의 패키지명, 핀맵, Motor Driver HAT 제어 IC, 토픽명은 가정하지 않았다.
- 구현 시작 시 `07_키트_구성품_활용_및_저장소_대조표.md`의 체크리스트로 기존 스켈레톤과 본 설계를 대조한다.
- 첨부 엑셀에는 Jetson 40핀 헤더의 일부 전원 핀 점검 결과가 불량으로 표시되어 있다. Jetson 헤더에서 모터·서보·센서 전원을 공급하지 않고 USB 및 별도 전원 레일을 사용한다.
- GA25-370 모터의 실제 정격전압·기어비·스톨전류와 지급 Motor Driver HAT의 제어 IC·최대전류는 2주차에 실물 라벨 및 전류 제한 전원으로 확인해야 한다.
- 추천 교체 시기는 장기간 열화 데이터가 없는 평가 단계에서 확정 예측값이 아니라 **규칙 기반 잠정치와 낮은/중간/높은 신뢰도**로 표시한다.
