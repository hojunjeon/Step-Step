# Triage Labels

| Canonical role    | Local status      | Meaning                        |
| ----------------- | ----------------- | ------------------------------ |
| `needs-triage`    | `needs-triage`    | 검토 및 분류가 필요함          |
| `needs-info`      | `needs-info`      | 추가 정보가 필요함             |
| `ready-for-agent` | `ready-for-agent` | AI 에이전트가 처리할 준비가 됨 |
| `ready-for-human` | `ready-for-human` | 사람의 작업이나 판단이 필요함  |
| `wontfix`         | `wontfix`         | 처리하지 않기로 결정함         |
